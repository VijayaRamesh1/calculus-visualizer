Accessibility Statement

Calculus Visualizer is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying relevant accessibility standards.

Measures to Support Accessibility

Calculus Visualizer takes the following measures to ensure accessibility:

  • Includes keyboard navigation for all interactive elements
  • Provides text alternatives for all non-text content
  • Implements ARIA landmarks and proper semantic HTML
  • Ensures sufficient color contrast throughout the application
  • Offers a "skip to main content" link for keyboard users
  • Supports zooming without loss of functionality
  • Works with screen readers through descriptive alt text and ARIA attributes
  • Provides alternative ways to access mathematical visualizations

Compatibility with Browsers and Assistive Technologies

Calculus Visualizer is designed to be compatible with the following assistive technologies:

  • Screen readers (including NVDA, JAWS, and VoiceOver)
  • Zoom settings in browsers up to 200%
  • Keyboard-only navigation
  • Speech recognition software
  • Screen magnification tools

Alternate Formats for Visualizations

Since our primary focus is visual representation of calculus concepts, we provide the following alternatives:

  • Text-based descriptions of all visualizations
  • Tabular data representation where applicable
  • Audio descriptions of key visualization changes
  • Mathematical formulas in accessible formats

Limitations and Alternatives

Despite our efforts, some content might not be fully accessible. In these cases, we offer:

  • Text-based alternatives to complex visual simulations
  • Downloadable resources in accessible formats
  • Direct assistance via our contact page

Feedback and Contact Information

We welcome your feedback on the accessibility of Calculus Visualizer. Please let us know if you encounter any accessibility barriers:
